# GCD:
GCD is Greatest Common Divisior is known as HCF.
It is Find by Euclid's Algorthim.

## Formula:
Higher_number = times*Lower_number + reminder

## Example:
```python
 Higher_number=64
 Lower_number = 48
 
 64 = 1*48+16
 #Then shift Higher_Number=Lower_Number:
 48 =3*16 +0
 16 = 3*0
```
Then GCD = 16
